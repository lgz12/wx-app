## 项目运行
>克隆项目：
>>git clone `'https://github.com/lgz12/wx-app.git'`

>需要下载并安装微信开发者工具
>>https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
>>自行下载

>打开微信开发者工具,打开项目文件夹即可

该小程序用的是本地服务器，后台用php,可根据各人喜好更改，这里不多做解释。
